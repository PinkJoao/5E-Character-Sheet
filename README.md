# 5E-Character-Sheet

## Project Overview
This open source project aims to provide a comprehensive and user-friendly digital character sheet for the Dungeons and Dragons 5th Edition role-playing game. Its primary purpose is to streamline character management, making it easier for players to create, track, and manage their characters. The application is designed to be intuitive and accessible, catering to both new and experienced players.

## Implemented Features
- **Basic Character Information:** Management of core stats, name, race, class, background, and alignment.
- **Ability Scores:** Input and calculation of ability scores and modifiers.

## Data Sourcing from 5e.tools Community
This project utilizes data generously provided by the open-source community of 5e.tools. The data is retrieved and processed to populate the character sheet with information such as spells, items, and features. It is important to clarify that this project does not host any content directly from 5e.tools, nor is it an official product of 5e.tools. I am immensely grateful to the 5e.tools community for their invaluable contributions to the open-source community, which have greatly enriched this project.
